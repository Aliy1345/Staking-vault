# Staking Vault DeFi Contract

A simple token staking dApp built on Ethereum using Solidity.

## What it does
Users deposit STK tokens and earn rewards over time.

## Tech Stack
- Solidity ^0.8.20
- Hardhat
- OpenZeppelin Contracts
- Ethereum Sepolia Testnet

## Deployed Contracts (Sepolia Testnet)
- StakingToken: `0xYOUR_TOKEN_ADDRESS`
- StakingVault: `0xYOUR_VAULT_ADDRESS`

## Setup
```bash
npm install
npx hardhat compile
npx hardhat test
